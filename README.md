# PP2_projeto_aquaPower

# Sistema de Controle de Consumo de Água e Energia.

Projeto desenvolvido para a disciplina de Projeto e Prática 2.

Seu principal objetivo é auxiliar os usuários no monitoramento de energia e água permitindo que eles cadastrem seus gastos, acompanhem seu histórico de consumo e visualizem suas informações através de gráficos.
